# 算命学 命式計算ロジック

## 概要

`/sanmeis/meishiki` で生年月日を入力し「計算する」ボタンを押すと、新しいウィンドウに命式（陰占・陽占・天中殺・五行）が表示される。計算ロジックは全てクライアントサイドのJavaScript（ビュー内に約4000行）で実装されている。

---

## 入力

| 項目 | 説明 |
|------|------|
| 姓 | テキスト |
| 名 | テキスト |
| 性別 | 男 / 女（大運の順行・逆行に影響） |
| 生年月日 | 年号（明治〜令和）+ 年 + 月 + 日 |

## 計算フロー

### Step 1: 節入り日の判定

生年月日が月の「節入り日」より前か後かを判定する。節入り日より前の場合、前月として扱う。

```
節入り日 = calc_setsu(年, 月)
```

年ごとに12ヶ月分の節入り日テーブル（1873年〜2050年）を持つ。

### Step 2: 日干支番号の算出

1912年2月1日からの経過日数を計算し、60で割った余りから日干支番号を求める。

```
経過日数 = (生年月日) - (1912/2/1)
日干支番号 = 経過日数 % 60（補正あり）
```

### Step 3: 年柱・月柱・日柱の干支計算

#### 年柱（年干・年支）
```
年干支番号 = (西暦年 - 3) mod 60
年干 = 年干支番号 mod 10
年支 = 年干支番号 mod 12
```
※ 2月の節入り前は前年扱い

#### 月柱（月干・月支）
```
月干支番号 = (西暦年×12 + 月 - 47) mod 60
月干 = 月干支番号 mod 10
月支 = 月干支番号 mod 12
```

#### 日柱（日干・日支）
```
日数 = cdate(年, 月, 日)  // ユリウス日ベースの計算
日干支番号 = 日数 mod 60
日干 = 日干支番号 mod 10
日支 = 日干支番号 mod 12
```

### Step 4: 蔵干の算出

各地支に対して、節入りからの経過日数に基づいて蔵干を決定する。

```
蔵干 = calc_zokan(地支, 節入りからの日数)
```

十二支ごとに蔵干の切り替え日数テーブルを持つ。各地支には1〜3個の蔵干がある。

### Step 5: 天中殺の算出

日干支番号から天中殺を計算する。

```
天中殺 = 11 - floor((日干支番号 - 1) / 10) × 2
```

天中殺は2つの連続する十二支で表される（例: 辰巳天中殺）。

### Step 6: 十大主星の算出

日干と他の干（年干・月干・蔵干）の関係から十大主星を求める。

```
主星 = calc_10shu(日干, 対象の干)
```

| 位置 | 計算 |
|---|---|
| 中央星 | 日干 × 月蔵干 |
| 東方星 | 日干 × 年蔵干 |
| 西方星 | 日干 × 日蔵干 |
| 北方星 | 日干 × 年干 |
| 南方星 | 日干 × 月干 |

十大主星: 貫索・石門・鳳閣・調舒・禄存・司禄・車騎・牽牛・龍高・玉堂

### Step 7: 十二大従星の算出

日干と地支の関係から十二大従星を求める。

```
従星 = calc_12jyu(日干, 対象の地支)
```

| 位置 | 計算 |
|---|---|
| 頭上（右上） | 日干 × 年支 |
| 初年（右下） | 日干 × 月支 |
| 晩年（左下） | 日干 × 日支 |

十二大従星: 天報・天印・天貴・天恍・天南・天禄・天将・天堂・天胡・天極・天庫・天馳

各従星にはエネルギー値（1〜12）が設定されている。

### Step 8: 五行の計算

陰占の全ての干支（天干3つ + 蔵干）の五行属性をカウントする。

| 五行 | 対応する十干 |
|---|---|
| 木 | 甲・乙 |
| 火 | 丙・丁 |
| 土 | 戊・己 |
| 金 | 庚・辛 |
| 水 | 壬・癸 |

## 主要なデータテーブル

### 十干（d_10kan）
甲(1), 乙(2), 丙(3), 丁(4), 戊(5), 己(6), 庚(7), 辛(8), 壬(9), 癸(10)

### 十二支（d_12shi）
子(1), 丑(2), 寅(3), 卯(4), 辰(5), 巳(6), 午(7), 未(8), 申(9), 酉(10), 戌(11), 亥(12)

### 十大主星（d_10shu）
10×10のマトリクス（日干×対象干）で決定

### 十二大従星（d_12jyu）
10×12のマトリクス（日干×地支）で決定

### 蔵干テーブル
十二支ごとに、節入りからの経過日数に応じて蔵干が変わる

### 節入り日テーブル
1873年〜2050年の各月の節入り日

---

## 異常干支の判定

日干支番号が特定の値の場合、「異常干支」としてピンク背景で表示される。

---

## 追加計算（大運・年運など）

命式の基本計算に加えて、以下も計算される：
- **大運**: 性別と年干の陰陽に基づき、順行/逆行で10年ごとの運勢を計算
- **年運**: 現在の年の干支から運勢を計算
- **位相法**: 干支同士の関係（合法・散法）を判定
- **身強/身弱**: エネルギー値の合計から判定

## 技術的特徴

- 全てクライアントサイドJavaScriptで計算（サーバーサイド処理なし）
- `window.open()` で新しいウィンドウに結果を描画
- HTMLテーブルを `document.write()` で動的生成
- 節入り日テーブルは1873年〜2050年をハードコード
